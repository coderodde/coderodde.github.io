BillPal 1.6

What is it?

BillPal is a simple graphical application for managing bills. You can create, edit and save your set of bills. It has nice coloring future which allow you to easily see the status of your bills.

Cost?
Using BillPal aint't cost a thing. Even better, it's open source.

Algotype.js - typesetting pseudocode in HTML files

Algotype.js?

This vanilla JavaScript library allows users to typeset pseudocode with custom XML tags and MathJax in HTML documents. The motivation wat that people (for the most part) don't need to take representation of pseudocode into account, but rather concentrate on the actual algorithms.

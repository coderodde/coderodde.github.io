Algotype.js - typesetting pseudocode in HTML files

Algotype.js?

This vanilla JavaScript library allows users to typeset pseudocode with custom XML tags and MathJax in HTML documents. The motivation was that people (for the most part) don't need to take representation of pseudocode into account, but rather concentrate on the actual algorithms.

Including Algotype.js

In order to load the Algotype.js library, all you need is: 

<link rel="stylesheet" type="text/css" href="algotype.css">
<link rel="stylesheet" type="text/css" href="algotype_my_config.css">
<script src="algotype.js"></script>
in the <head> of the HTML file.

Dependencies

Algotype.js implicitly loads MathJax library for writing TeX.

alg-algorithm

Summary
The tag alg-algorithm defines an entire algorithm.
Synopsis
<alg-algorithm name="Algorithm-Name" parameters="x A R" comment="Algorithm header comment">
...
</alg-algorithm>
The above will be rendered as follows

namethe name of the algorithmrequired
parametersthe list of algorithm's parametersrequired
commentthe algorithm commentoptional

Notes
  • Make sure that you wrap the element <alg-algorithm>...</alg-algorithm>; into a, say <div> that does not contain any other elements. Otherwise, it will be placed as the very last element of its parent element even if the algorithm is not the last element of the parent.
  • The parameter list accepts several formats. The above parameter list $(x, A, R)$ may be specified in parameters attribute as
      x A R
      x, A, R
      (x A R)
      (x, A, R)
    • You can also omit some commas in the above lists, or omit one or both parentheses.

alg-break

Summary
The tag alg-break typesets a break statement.
Synopsis
<alg-break >Label</alg-break>
The above will be rendered as follows at line 1:

label

commentthe break statement commentoptional

Notes
  • The token label is optional and is used most ofter for breaking out of an outer loop.
